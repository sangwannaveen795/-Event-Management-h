## Overview

This repository contains the front-end code for the **Hospitality Solutions** event management company, led by **Naveen Choudhary**. The website is designed to be visually appealing, user-friendly, and effectively showcase the company's premium wedding and corporate event management services, based on the provided company profile.

The design utilizes a luxury aesthetic with a **Navy Blue and Gold** color scheme, incorporating dynamic visuals and essential booking features.

## Key Features

The website incorporates innovative and user-friendly features, ensuring all components are functional and responsive:

* **Responsive Design:** Fully adapts to mobile, tablet, and desktop views.
* **Personalized Branding:** Features a dedicated "About Naveen Choudhary" section to build customer trust and connect with the founder.
* **Functional Buttons:** All primary Call-to-Action buttons (e.g., "Plan My Event," "Explore Services") and utility links ("Terms of Service," "Privacy Policy") are functional via smooth scrolling or interactive modals.
* **Intuitive Booking Form:**
    * Includes a **Date Picker** element for customers to specify their desired event dates.
    * Features a **File Upload** option for customers to submit references (e.g., wedding cards or event briefs).
* **User-Friendly Contact:** A floating **WhatsApp Button** is included for instant, direct communication, enhancing user satisfaction.
* **Service Showcase:** Detailed sections for core services like "Shadows," "Luggage Management," and "Logistics."
* **Creative Visuals:** Uses high-quality, relevant imagery and parallax effects to create an immersive, luxurious feel.

## Technology Stack

The website is a static HTML page, relying on core web technologies:

* **HTML5:** Structure and Semantics.
* **CSS3:** Styling, responsive design, and animations.
* **JavaScript (Vanilla JS):** Handled button responsiveness, modal functionality, and smooth scrolling.
* **External Assets:** FontAwesome (icons) and Google Fonts for enhanced typography.

## Setup and Running Instructions

Since this project consists of a single HTML file with embedded CSS and JavaScript, deployment is extremely simple.

1.  **Save the Code:** Save the provided HTML code into a file named `index.html`.
2.  **Open in Browser:** Double-click the `index.html` file or open it using any modern web browser (Chrome, Firefox, Edge).

The website will run instantly and locally on your machine.
